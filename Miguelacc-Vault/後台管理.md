# 🧑‍💼 後台管理

回到 [[00-總覽 (MOC)]]。

## 登入

| 項目 | 值 |
|------|-----|
| 登入網址 | http://localhost:3000/admin/login |
| 帳號 | `admin@example.com` |
| 密碼 | `admin1234` |

> [!note] 認證機制
> - 採用 JWT（`jose` 套件）+ HttpOnly Cookie（cookie 名稱 `admin_token`）
> - Session 有效時間 **8 小時**
> - 帳密來自 [[環境變數設定]] 的 `ADMIN_EMAIL` / `ADMIN_PASSWORD`
> - 比對使用 `timingSafeEqual`（防時序攻擊）

## 後台路由結構

```
/admin/login              登入頁
/admin/(dashboard)/...    登入後的管理介面
```

- 認證邏輯：`src/lib/auth.ts`
- 登入動作：`src/app/admin/(auth)/login/actions.ts`
- 導覽列：`src/app/admin/(dashboard)/AdminNav.tsx`

## 資料模型（可管理的內容）

來自 `prisma/schema.prisma`：

| 模型 | 說明 |
|------|------|
| `User` | 使用者（admin 帳號） |
| `Category` | 商品分類 |
| `Product` | 商品（名稱、描述、圖片、價格、蝦皮連結覆寫等） |
| `ProductVariation` | 商品規格 / SKU（價格、庫存） |
| `Order` | 訂單（客戶資訊、狀態、金額） |
| `OrderItem` | 訂單品項 |
| `Payment` | 付款紀錄（綠界交易、狀態） |

### 訂單狀態 `OrderStatus`
`PENDING` → `PAID` → `SHIPPED` ／ `FAILED` ／ `CANCELED`

### 付款狀態 `PaymentStatus`
`WAITING` → `SUCCESS` ／ `FAILED` ／ `REFUNDED`
